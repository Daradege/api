Daradege API Documentation

Welcome to the Daradege API documentation. This API provides various AI-powered services including text-to-speech conversion, image generation, and chat capabilities. Each endpoint is documented below with examples and required parameters.

All endpoints support both GET and POST methods. For GET requests, parameters should be passed as URL query parameters. For POST requests, parameters should be sent as JSON in the request body.
